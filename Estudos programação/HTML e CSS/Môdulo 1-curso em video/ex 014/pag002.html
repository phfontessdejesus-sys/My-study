Fonte das letras

Quando queremos algo em negrito, usamos <strong>

Quando queremos algo em italico, usamos <em>

Se queremos uma fonte menor, usamo <small>

Quando queremos deixar embaixo ou em cima da palavra letras ou números usamos <sub< ou <sup> respectivamente
Exemplo: H20 e x3+4

Podemos sublinhar as linhas usando <u>

Podemos colocar uma linha no meio com <del>

Para marcar algo como se fosse marca texto usamos o <mark>

Imagens

Para imagens basta digitar "img" e apertar enter
o camndo de "img" é <img src ="" alt = "">

Favicon (Imagem da guia)

Para mudar a imagem da guia, basta colocar em cima do <title> <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">,

Listas

Para a criação de listas usamos o <ol> para listas ordenadas, usamos listas também para <ul> para listas desordenadas

Agora o <li> usamos para colocar o item da listas

Para você colocar o tipo de classificação de cada item da lista ( A, a 1, X) usamos o <ol ou ul type = "">

Para você fazer uma sub lista basta colocar <ol> dentro do <li>

Caso a gente queria que a lista comece em um número ou letra especifica usamos <ol type ="" start = "">
